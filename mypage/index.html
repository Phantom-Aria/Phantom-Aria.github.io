CSS的四种引入方式

内联样式(直接写进html元素,仅对当前的html元素有效,上面的h1标签没变)

内部样式表,需要自己写入css规则,注意style标签写在head中,我这里改了h2的color,上面h2标签的内容颜色就变了

外部样式表,需要在head中引入link:css,在外部写css规则,href引入

导入样式表，可以使用 @import 规则在一个 CSS 文件中引入另一个 CSS 文件。例如，在 styles.css 文件中引入 common.css 文件

@import url("common.css");

这种方式与外部样式表类似，但需要注意 @import 规则必须位于 CSS 文件的顶部。

css是由优先级的

  • 相同的规则按照加载顺序，写在后面的优先级更高
  • 继承的样式优先级低于直接指定的
  • 内联样式优先级最高，除了!important
  • 类型选择器小于类选择器（class）小于ID选择器

盒子模型

盒子模型重要的属性由三个

  • 边框 border 边框宽度，类型，颜色
  • 填充部分 padding 上下左右填充的距离，也可以单独指定padding-top这样的
  • 外边距 margin
  • 后两个可以直接指定4个方向（按照顺时针）
